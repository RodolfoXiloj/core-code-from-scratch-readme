  &lt;1&gt;

  &lt;/1&gt;
  &lt;2&gt;

  &lt;/2&gt;
  &lt;3&gt;
    Divide the result obtained in step 2 by 1.8
  &lt;/3&gt;
  &lt;4&gt;
    The resulting value is the temperature in Celsius
  &lt;/4&gt;
