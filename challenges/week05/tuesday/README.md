<h3>'IF' statement in JavaScript 📝</h3>

```js
let x = 0
if (x > 5){
	console.log("x is greater than 5");
}
```

</br>

<h3>WHILE' loop in JavaScript 📝</h3>

```js
let i = 0;
while (i < 10){
	console.log(i);
	i++;
}
```
</br>

<h3>'FOR' loop in JavaScript 📝</h3>

```js
for(let i = 0; i < 10; i++){
	console.log(i);
}
```
